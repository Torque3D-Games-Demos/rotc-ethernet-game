Controls reference

Basic Controls

esc Toggle in-game menu
c Toggle commander view
t Team chat
z Global chat

Observer Controls

W A S D Movement
Mouse X & Y axis Look
Mouse Z axis (wheel) Zoom
r Zoom
y Show bigger minimap

Etherform Controls

W A S D Movement
Space / Control Up / Down Movement
Mouse X & Y axis Look
Mouse Z axis (wheel) Zoom
Tab Manifest as CAT
1 Select primary weapon: Micro-Grenade Launcher
2 Select primary weapon: Hyper-Missile Launcher
q Select primary weapon: Toggle between weapons
r Zoom
y Show bigger minimap

CAT Controls

W A S D Movement
Mouse X & Y axis Look
Mouse Z axis (wheel) Zoom
Left mouse button Fire equipped weapon
Middle mouse button Use Crystal Sceptre (Defensive)
Right mouse button Use Crystal Sceptre (Offensive)
Tab Return immediately to Etherform
Left shift Walk slowly
Left control Use Etherboard
1 Equip Trident
2 Equip primary weapon
q Toggle equipped weapon
r Zoom
y Show bigger minimap

Commander view Controls

W A S D Pan map
Mouse Z axis (wheel) Zoom
q Zoom out
e Zoom in
c Deactivate commander view
